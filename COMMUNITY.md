# Community Guidelines

Welcome to the redis-distro-scheduler community!

Our mission:  
Provide a robust, production-grade distributed scheduler that is safe to run across Kubernetes pods with Redis locking and multi-tenant guardrails.

## Ways to Participate

- Ask & answer questions
- Report bugs
- Improve documentation
- Submit PRs
- Share use cases from your company

## Values

- Be respectful
- Help newcomers
- Focus on growth & learning
- Prefer clarity over cleverness

## Communication Channels

- GitHub Issues
- GitHub Discussions
- PR review threads
